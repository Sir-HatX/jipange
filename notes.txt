API Design:
- Budget:
  - Name i.e Oct 2024 Budget
  - start_date
  - end_date
  - total_amount
  - allocation_rule i.e Needs 50%, Wants 20%, Saving 30%
  - status i.e open | closed
  - Budgeted items:
    - Name i.e Transport
    - Description
    - Amount i.e 2000
    - Category i.e Want
    - Tags

- Expenses:
  - Description
  - amount
  - trx_type [cash | mpesa | bank]
  - budgeted_item_id
  - trx date
  - trx_charges
  - Tags

- Accounts:
  - Account_name
  - Account_type
  - Account_balance
  
- Transaction:
  - source_account
  - destination_account
  - amount
  - trx_cost
  - trx_date
